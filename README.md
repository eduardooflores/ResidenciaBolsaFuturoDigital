<h1>Residência Bolsa Futuro Digital</h1>

<h2>Propósito</h2>
<p>
O programa <strong>Bolsa Futuro Digital</strong> tem como objetivo formar profissionais capacitados no 
desenvolvimento de soluções tecnológicas inovadoras.
</p>
<p>
Este repositório reúne os projetos e trabalhos realizados durante a residência, 
servindo como portfólio e registro da evolução ao longo do programa.
</p>

<h2>Tecnologias</h2>
<p>O programa ensina e utiliza as seguintes tecnologias e conceitos:</p>
<ul>
  <li><strong>Python 3</strong></li>
  <li><strong>Django</strong></li>
  <li>WebServices</li>
  <li>Lógica de Programação</li>
  <li>Orientação a Objetos (OOP)</li>
  <li>Banco de Dados Relacionais</li>
  <li>Padrões de Desenvolvimento de Software</li>
</ul>

<h2>Estrutura do Repositório</h2>
<ul>
  <li>Lista de Exercícios - Algoritmos sequenciais | Arquivos com exercícios de lógica</li>
  <li>Lista de Exercícios - Estruturas de decisão | Arquivos com exercícios com estruturas de decisão</li>
  <li>Lista de Exercícios - Estruturas de Repetição | Arquivos com exercícios de estruturas de repetição</li>
  <li>Lista de Exercícios Sequenciais - Definições | Arquivos com exercícios elaborados Puramente Sequenciais</li>
  <li>Lista de Exercícios Condicionais - Definições | Arquivos com exercícios elaborados sobre Decisão</li>
  <li>Atividade OO | Atividade Orientação a Objetos</li>
  <li>DjangoProjeto | Códigos e conceitos ensinados durante a residência</li>
  <li>ProjetoLocadora | Desafio de implementação UML com models do Django</li>
  <li>meu_ingresso | Desafio de implementação Django com relacionamentos</li>
  <li>relacionamentos | Exemplos de relacionamentos utilizando Django ensinado durante a residência</li>
  <li>youtube | Desafio de engenharia reversa do youtube | Importando arquivos e utilizando Managers para consultas</li>
  <li>techtinga | Desafio final da residência | Implementação de todos os conceitos juntos | models, relacionamentos, webservices, managers, RESTful API, normalização de dados e segurança com tokens</li>
</ul>
<p>Em cada <strong>Lista de Exercícios </strong> é possível encontrar um arquivo PDF com os exercícios propostos.</p>
<p>Nos arquivos <strong>ProjetoLocadora</strong>, <strong>meu_ingresso</strong>, <strong>youtube</strong> e <strong>techtinga</strong> é possível encontrar arquivos .csv e diagramas UML para compreender melhor o desafio proposto.</p>
